### 🔇 MUTE Control
- **Pin:** `GPIOA10`
- `Low`  → Có nhạc  
- `High` → Không có nhạc  

---

### 🎛 KEY Control
- **Pin:** `GPIOA23`
- **Vị trí:** Điểm nối giữa tụ **C15** và điện trở **104**

#### 🔧 Chế độ hoạt động

- 🔌 **Key OFF + Cắm AUX**  
  → Tự động chuyển sang **AUX Mode**

- 🔘 **Key ON**  
  → Hỗ trợ **11 phím Key**
